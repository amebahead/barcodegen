# 바코드 / QR 멀티 생성기

여러 개의 데이터(주문번호·상품코드·URL 등)를 한 번에 입력하면, 각각을 **바코드** 또는 **QR 코드**로 만들어 한 화면에 격자로 보여주는 웹 도구입니다.

## 🔗 바로 사용하기

**👉 https://amebahead.github.io/barcodegen/**

설치도, 회원가입도 필요 없습니다. 위 주소를 열면 바로 사용할 수 있어요. (PC·모바일 모두 가능)

## ✨ 이런 걸 할 수 있어요

- **한 번에 여러 개** — 한 줄에 하나씩 입력하면, 입력한 개수만큼 코드가 자동으로 만들어집니다.
- **바코드 ⇄ QR 전환** — 버튼 하나로 전체를 CODE128 바코드 또는 QR 코드로 바꿀 수 있습니다.
- **입력하면 바로 표시** — '생성' 버튼을 누를 필요 없이, 입력하거나 붙여넣으면 즉시 격자에 반영됩니다.
- **코드 아래 원본 텍스트** — 각 코드 밑에 원래 값이 적혀 있어 무엇인지 한눈에 알 수 있습니다.
- **화면에 맞춰 자동 정렬** — 창 크기에 따라 한 줄에 들어가는 개수가 자동으로 조절됩니다.
- **오류는 그 칸만** — 만들 수 없는 값이 섞여 있어도 해당 칸만 표시되고, 나머지는 정상 생성됩니다.

## 🖱️ 사용 방법

1. **https://amebahead.github.io/barcodegen/** 접속
2. 위쪽 입력창에 **한 줄에 하나씩** 데이터를 입력 (엑셀·메모장에서 복사해 붙여넣어도 됩니다)
3. **바코드 / QR** 중 원하는 형식을 선택
4. 아래 격자에 생성된 코드를 확인
5. 필요하면 화면을 캡처하거나 인쇄해서 사용하고, 휴대폰 카메라로 스캔해 값이 맞는지 확인할 수 있습니다

## 👍 편리한 점

- **설치 불필요** — 브라우저만 있으면 됩니다.
- **안전** — 입력한 데이터를 외부로 전송하지 않습니다. 코드 생성이 전부 내 브라우저 안에서 이뤄지므로 내부 코드나 민감한 값도 안심하고 쓸 수 있습니다.
- **무료** — 사용에 비용이 들지 않습니다.
- **대량 작업에 강함** — 목록을 통째로 붙여넣으면 수십 개도 한 번에 생성됩니다.

## 💡 활용 예시

- 상품·재고 라벨, 자산 관리 스티커
- 행사 입장·이벤트용 QR 코드 묶음
- URL 모음을 QR로 변환
- 송장·주문번호 일괄 바코드화

## 📝 참고

- **바코드(CODE128)** 는 영문·숫자·기호를 지원합니다. 한글처럼 지원하지 않는 문자가 들어가면 그 칸만 "생성 실패"로 표시됩니다.
- **QR 코드** 는 한글, URL 등 사실상 모든 텍스트를 담을 수 있습니다.

## 개발자용

브라우저에서 동작하는 정적 웹앱입니다. 별도 서버가 없으며, 바코드는 [JsBarcode](https://github.com/lindell/JsBarcode), QR은 [qrcode-generator](https://github.com/kazuhikoarase/qrcode-generator)로 생성합니다. (TypeScript + Vite)

### 개발

```bash
npm install
npm run dev      # 개발 서버
npm test         # 단위 테스트
npm run build    # dist/ 정적 빌드
npm run preview  # 빌드 결과 미리보기
```

### 배포 (GitHub Pages)

`main` 브랜치에 push하면 GitHub Actions가 자동으로 빌드 후 Pages에 배포합니다.

1. 저장소 **Settings → Pages → Source = "GitHub Actions"** 선택
2. `main`에 push → 자동 배포 → `https://<사용자>.github.io/barcodegen/`

> 저장소 이름이 `barcodegen`이 아니면 `vite.config.ts`의 `base`를 `'/<저장소명>/'`로 변경하세요.
