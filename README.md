# Barcode / QR Code Generator

여러 값을 한 번에 **바코드·QR 코드**로 만들어 격자로 보여주는 웹 도구입니다.
설치·가입 없이 브라우저에서 바로 사용하세요.

### 👉 https://amebahead.github.io/barcodegen/

## 기능

- **한 번에 여러 개** — 한 줄에 하나씩 입력하면 입력 즉시 격자로 생성 (생성 버튼 없음)
- **Barcode(CODE128) ⇄ QR Code** 전환
- 각 코드 아래 **원본 값 표시**, 화면 크기에 맞춰 **자동 정렬**
- 잘못된 값은 **해당 칸만** 오류 표시 (나머지는 정상)
- **내보내기** — 칸별 **PNG** · 전체 **ZIP** · **Print**(인쇄창에서 "PDF로 저장" 가능)

## 사용법

1. 위 링크 접속
2. 입력창에 값을 한 줄에 하나씩 입력 (엑셀·메모장 복붙 OK)
3. **Barcode / QR** 선택
4. 칸의 **PNG**, 상단 **Download all (ZIP)** 또는 **Print** 로 내보내기

> 모든 처리는 브라우저 안에서 이뤄져 입력값이 외부로 전송되지 않습니다.
> Barcode(CODE128)는 영문·숫자·기호용이며, 한글 등은 QR을 사용하세요.
